# Project SimpCar

Project SimpCar is a 3D car customization application built with Panda3D that allows you to personalize your car with various customization options.

## Features

- **Wheel customization**: Rim and tire selection, color, diameter, width, camber, offset, and toe adjustment
- **Car customization**: Color changes, body kits, and individual body part modifications
- **Ground selection**: Multiple surface options
- **Scalable content system**: New cars, wheels, and grounds can be added without code changes

## Download Instructions

This repository contains the project split into 3 parts to respect file size limits:

- **project simpcar part1.zip**: Code files, wheels, cubemaps, fonts (7.19 MB)
- **project simpcar part2.zip**: Selected grounds, UI images (20.19 MB)
- **project simpcar part3.zip**: Additional grounds, GitHub images (23.17 MB)

### To Install:

1. Download all 3 zip files
2. Extract them to the same directory
3. Run `main.py` with Python 3.11+

## Dependencies

```bash
pip install typing-extensions
pip install panda3d==1.10.13.post1
pip install panda3d-gltf==1.1.0
pip install panda3d-simplepbr==0.11.2
```

## Shortcuts

- `A`: Toggle camera autorotation
- `B`: Body shop menu
- `G`: Garage menu
- `I`: Save screenshot
